== Description ==
The SR Post Like Dislike Plugin is a powerful and easy-to-use plugin that adds a like and dislike functionality to your WordPress posts, pages, and custom post types. This plugin allows your users to interact and give feedback on your content, making it more engaging and interactive.

== Installation ==
Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress


== Requirements ==
This plugin requires:
At least WordPress 5.0
Tested up to WordPress 6.1.1
Requires PHP 7.2 or higher
WC requires at least 3.1 and WC tested up to 7.2.3


== Frequently Asked Questions ==
= How can I use this plugin on my website?
You can use this plugin by installing it and activating it on your website. Once it's activated, you can go to the settings of the plugin to configure it as per your needs.
= What are the requirements for this plugin to work?
This plugin requires WordPress 5.0 or higher, PHP 7.2 or higher and WC 3.1 or higher.
= Can this plugin be used for custom post types?
Yes, this plugin can be used for custom post types as well.
